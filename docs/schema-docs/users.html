III-a

users

재사용
계정

계정 마스터. 아바타는 기존 profile_image_url 재사용(→ avatarUrl). 표시이름 name(→ displayName). deleted_at 소프트 삭제.

컬럼타입제약설명
idbigintPKautoincrement
emailvarchar(255)UK*활성 행 부분 유니크. OAuth 이메일
namevarchar(50)표시 이름 → API displayName
biovarchar(255)소개 (선택)
location_labelvarchar(100)위치 라벨 (선택)
profile_image_urlvarchar(500)avatar 재사용→ API avatarUrl. URL 참조만 (업로드 금지)
taste_vectorhalfvec(512)halfvec취향 벡터. ivfflat halfvec_cosine_ops
created_attimestamptzDEFAULT now()생성 시각
deleted_attimestamptz소프트 삭제 시각 (NULL = 활성)

인덱스

  • idx_users_email_active — UNIQUE(email) WHERE deleted_at IS NULL
  • idx_users_taste_vector — ivfflat (halfvec_cosine_ops)

참조됨

user_auth_providers · user_consents · user_saved_spots (모두 CASCADE)

III-b

user_auth_providers

변경 (컬럼 드롭)
OIDC 매핑

OIDC 제공자 ↔ 유저 upsert 매핑. refresh_token_enc 드롭 (refresh는 서버 미저장).

컬럼타입제약설명
idbigintPKautoincrement
user_idbigintFK→ users.id · ON DELETE CASCADE
providervarchar(16)NOT NULLCHECKkakao · google · apple (email은 레거시)
provider_user_idvarchar(255)NOT NULLOIDC subject (sub)
linked_attimestamptzDEFAULT now()연결 시각

인덱스 · 키

  • UNIQUE (provider, provider_user_id)
  • idx_auth_providers_user — (user_id)

드롭된 컬럼

refresh_token_enc — Redis jti 전용 결정과 모순되는 죽은 컬럼

user_consents

변경 (컬럼 드롭)
동의 스냅샷

동의 스냅샷. 버전은 terms_version. notification_consent 드롭.

컬럼타입제약설명
user_idbigintPKFK→ users.id · ON DELETE CASCADE
location_consentbooleanDEFAULT false위치 동의 (거리칩 활성 조건)
photo_consentbooleanDEFAULT false사진 동의
terms_versionvarchar(16)NOT NULL동의한 약관 버전 (예: v1.0)

드롭된 컬럼

notification_consent — 알림 비목표
